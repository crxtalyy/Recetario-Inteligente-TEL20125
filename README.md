# Recetario Inteligente

## Descripción
Proyecto de software desarrollado en C++ utilizando el framework Qt. El **Recetario Inteligente** permite la gestión, almacenamiento y escalado dinámico de recetas de cocina, optimizando la logística y presupuestos para eventos sociales o vida universitaria.

## Características Principales
* **Gestión de Recetas:** Creación, lectura y almacenamiento local de recetas.
* **Escalabilidad Automática:** Ajuste dinámico de cantidades según el número de personas.
* **Análisis Nutricional:** Visualización de aportes nutricionales mediante gráficos circulares (QtCharts).
* **Seguridad:** Registro de usuarios con contraseñas cifradas.

## Estructura del Proyecto
* `/src`: Código fuente completo del proyecto.
* `/docs`: Documentación técnica (Análisis, Manual de Usuario y de Instalación).

## Tecnologías Utilizadas
* **Lenguaje:** C++ (Estándar 17)
* **Framework:** Qt 6.x / 5.x
* **Gráficos:** QtCharts

## Instalación
Para instrucciones detalladas sobre cómo compilar y ejecutar el proyecto, por favor consulta nuestro [Manual de Instalación](docs/Manual_Instalacion.pdf).

1. Clona el repositorio: `git clone https://github.com/tu-usuario/Recetario-Inteligente.git`
2. Abre el proyecto en **Qt Creator** seleccionando el archivo `CMakeLists.txt`.
3. Configura el Kit de compilación y presiona el botón **Build**.
